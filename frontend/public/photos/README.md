# Photos folder — COMPASS

Drop your photos into **this folder** and the app will pick them up automatically.

## File names the app looks for

| File name            | Where it appears                                                |
| -------------------- | --------------------------------------------------------------- |
| `hero.jpg`           | Front-page hero photograph (right side of the masthead)         |
| `vidarbha.jpg`       | Geography card for Vidarbha (entry screen step 3)               |
| `odisha.jpg`         | Geography card for Coastal Odisha (entry screen step 3)         |
| `ladakh.jpg`         | Geography card for Ladakh (entry screen step 3)                 |
| `sol-biodyke.jpg`    | Solution card — Bio-Dyke + Mangrove Buffer                      |
| `sol-watershed.jpg`  | Solution card — Watershed Treatment                             |
| `sol-icestupa.jpg`   | Solution card — Ice Stupa                                       |
| `sol-sri.jpg`        | Solution card — System of Rice Intensification                  |
| `sol-heatplan.jpg`   | Solution card — Village Heat Action Plan                        |
| `sol-ews.jpg`        | Solution card — Cyclone Early Warning Cascade                   |
| `sol-pmfby.jpg`      | Solution card — PMFBY Claim Navigation                          |
| `sol-solarhouse.jpg` | Solution card — Solar-Passive House Retrofit                    |

## Notes

- JPG works best. PNG also works — just rename the file to `.jpg` or update the data file.
- Recommended size: **1600×1200px** (4:3) or larger. The app will crop/fit automatically.
- If a file is missing, the app shows a **beautifully styled placeholder** — nothing breaks.
- If a file is missing, the app also tries a Pexels fallback image as a safety net.
